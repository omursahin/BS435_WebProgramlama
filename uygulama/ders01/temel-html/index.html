Anasayfa

WebStorm üzerinden sağ tık, Open in Browser komutları ile çalıştırabilirsiniz. Bir web server başlatacak ve klasördeki dosyaları HTTP üzerindeki geçici bir port üzerinden yayınlayacaktır.

Dosyaları açmak için bir diğer seçenek de doğrudan dosyayı dosya sistemi üzerinden açmaktır. Böyle açtığınız zaman http yerine file protokolü üzerinden erişilecektir. Ancak sunucu üzerinden bu dosyaların yayınlanması uygulamadaki genel yaklaşımdır.
